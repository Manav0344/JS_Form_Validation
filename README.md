## Project Overview
This is a **JavaScript Form Validation project** designed to help users understand and implement **client-side validation** using **HTML, CSS, Bootstrap, and JavaScript**.  
The project allows users to **enter details in a registration form** and validates the inputs before submission.  

It is a simple, interactive, and responsive form designed for educational purposes and practical learning of **front-end validation techniques**.

---

## Features
- Responsive **registration form** using **Bootstrap**  
- Real-time **form validation** feedback with colors and error messages  
- Validation rules for all form fields:  
  - Name  
  - Email  
  - Password & Confirm Password  
  - Phone number  
- Success message after successful submission  
- Modern **UI design** with custom CSS styles  

---

## Technologies Used
- HTML5  
- CSS3  
- Bootstrap 5  
- Vanilla JavaScript

Form Validation Rules
Field	Validation Rule
Full Name	Must contain at least 3 characters
Email	Must be in a valid email format (example@domain.com
)
Password	Must be at least 6 characters
Confirm Password	Must match the Password field
Phone Number	Must be exactly 10 digits
